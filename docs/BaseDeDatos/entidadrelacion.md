# Diagrama ER — Modelo Entidad-Relación (Conceptual)
## Descripción

Este diagrama representa el modelo entidad-relación conceptual del sistema.
Muestra entidades principales, atributos y relaciones sin entrar en detalles físicos.
El objetivo es comprender el dominio y sus vínculos.

![Diagrama ER — Modelo Entidad-Relación (Conceptual)](/img/Modelo-entidad-relacion.drawio.png)

## Componentes involucrados

- Usuario / Usuarios pendientes
- Colorimetría
- Makeup
- Outfit
- Producto
- Tienda
- Cosmetiquero
- Influencer
- Mensaje

## Consideraciones técnicas

- Define relaciones 1:1, 1:N y N:M.
- Refleja reglas de negocio.
- Sirve como base para el modelo relacional.
- Facilita comunicación con stakeholders.

## Observaciones

- No incluye tipos ni claves físicas.
- Es ideal para documentación y diseño inicial.
- Se alinea con DDD.
